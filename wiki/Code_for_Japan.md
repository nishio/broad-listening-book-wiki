# Code for Japan

日本のシビックテックNPO。[[関治之]]が代表。市民参加プラットフォーム[[Decidim]]のパートナーシップ展開やシビックテック活動を推進。

## 詳細

関治之は2024年5月23日に[[安野貴博]]に[[Talk_to_the_City]]を紹介し、これが安野のブロードリスニング活用のきっかけとなった。2024年のAI社会カンファレンスで「シビックテックによる社会と民主主義のアップデート」と題する基調講演を行っている。

Plurality Tokyoの共同主催者でもある。
