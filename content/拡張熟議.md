# 拡張熟議

ICT（情報通信技術）を活用して、[[熟議民主主義]]の規模・頻度・形式を拡張する概念。従来の小規模対面議論を、AIの支援により数千〜数万人規模のオンライン議論へ拡大する。

## 詳細

拡張熟議には3つの拡張軸がある：
- **規模の拡張** — 数十人から数千〜数万人へ
- **頻度の拡張** — 単発のイベントから継続的な対話へ
- **形式の拡張** — 対面からオンライン・非同期へ

サイクルとしては「収集 → 構造化 → 熟議 → 意思決定 → 説明 → 再収集」のループを回す。量と質のトレードオフが存在し、大規模収集（スケールアップ）と対話品質の維持（スケールアウト）のバランスが課題となる。
